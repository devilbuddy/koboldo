# koboldo

Rust game engine hobby project (based on rust-sdl2 - https://github.com/AngryLawyer/rust-sdl2)

Features:

* Main loop
* Sprites (with animation frames)
* Bitmap fonts (bmfont - http://www.angelcode.com/products/bmfont/doc/file_format.html)
* Ninepatch
